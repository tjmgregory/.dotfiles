$ unikraft resource delete --help

stdout:
	Remove a resource.

	Usage:
	  unikraft resources delete [<target> ...] [flags]

	Arguments:
	  [<target> ...]
	          Target resources to remove.

	Fields:
	  type
	  key

	Flags:
	      --all
	          Remove all resources. Prompts for confirmation.
	      --filter
	          Filter resources to remove (e.g. --filter state==running). Prompts for confirmation.
	      --force
	          Do not prompt for confirmation when using --all or --filter.
	  -f, --field
	          Specify which fields to include in the output.
	  -o, --output
	          Output format. One of: kv, table, json, yaml, raw, quiet, template.
